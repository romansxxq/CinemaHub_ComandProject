## 🔧 ЯК ВИКОРИСТОВУВАТИ

### 🎯 Для новачків
1. Прочитайте **QUICK_START.md**
2. Встановіть залежності
3. Запустіть `pytest tests/ -v`
4. Дивіться вивід

### 📚 Для розробників
1. Відкрийте **TESTING.md**
2. Вивчіть структуру фіксчур в `conftest.py`
3. Подивіться приклади тестів
4. Напишіть свої тести

### 🔍 Для тестувальників
1. Запустіть `pytest tests/ --cov=api --cov-report=html`
2. Дивіться `htmlcov/index.html` для звіту
3. Перевіряйте покриття коду

### 🤖 Для CI/CD
1. Інтегруйте в GitHub Actions
2. Запускайте при кожному PR
3. Вимагайте мінімум 80% покриття
